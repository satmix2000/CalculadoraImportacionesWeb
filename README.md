# Calculadora de Importaciones Web

Aplicación web para cálculo de importaciones China → Argentina.

## Características
- Conversión CNY → USD → ARS con APIs en tiempo real
- Cálculo automático de aranceles, impuestos y costos logísticos
- Integración con costos de MercadoLibre
- Análisis de rentabilidad con margen objetivo 35%

## Tecnologías
- ASP.NET Core Blazor Server
- Entity Framework Core
- SQL Server / SQLite
- Bootstrap 5
